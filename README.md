Sample example to setup a NodeJS server on Heroku

#### Start Application

    $ yarn dev

### Build For Production

    $ yarn build

### Run locally in production mode

    $ node ./build/main.js

## Heroku

### Create A Heroku Instance

    $ heroku create

### Deploy To Heroku

    $ git push heroku master
